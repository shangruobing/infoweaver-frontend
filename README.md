# NFQA前端开发

#### 介绍

NFQA前端部分，采用Vue3和Element-Plus组件库编写，实现聊天机器人和通知文件管理系统

#### 主要依赖

- Vue 3.2.25
- TypeScript 4.6.2
- Element-Plus 2.0.1
- vue3-beautiful-chat 3.2.0
- docx-preview 0.1.8


#### 安装教程

1.  安装 npm install
2.  运行 npm run dev
3.  构建 npm run build

#### 使用说明

1.  准备好通知文件(docx)格式
2.  启动Neo4j服务
3.  使用数据库端的ipynb文件将数据导入数据库(MySQL和Neo4j)
4.  启动Django服务器
5.  使用Vite打包启动Vue
